# Galton–Watson Branching Processes: Simulation, Visualization and Reduced Processes

Данный проект представляет собой интерактивное приложение на Python для изучения ветвящихся процессов Гальтона–Ватсона. Программа позволяет:
- задавать распределение числа потомков (вручную или выбирая стандартные: Пуассона, геометрическое, детерминированное);
- строить график производящей функции \( S(x) \) и лестницу Ламерея;
- анимировать итерации;
- симулировать одну или множество траекторий процесса;
- вычислять теоретические моменты;
- анализировать **редуцированные процессы** (по главе 10 курса В.А. Ватутина) – находить расстояние до ближайшего общего предка и проверять предельные теоремы.

## Требования

- Python 3.7 или выше
- Библиотеки: `numpy`, `matplotlib`, `scipy`, `tkinter` (обычно входит в стандартную поставку Python)
